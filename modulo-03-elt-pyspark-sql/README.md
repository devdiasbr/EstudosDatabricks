## Objetivos de Aprendizado

- [ ] Ler dados em múltiplos formatos (CSV, JSON, Parquet, Delta)
- [ ] Aplicar transformações com DataFrames e SQL
- [ ] Diferenciar Actions vs Transformations (lazy evaluation)
- [ ] Criar e usar Temp Views e Global Temp Views
- [ ] Implementar um pipeline ELT completo (Bronze → Silver → Gold)
- [ ] Usar higher-order functions e pivot
